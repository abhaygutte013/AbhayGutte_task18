# Interactive Greeting Web Page

## Project Title

Interactive Greeting Web Page Using HTML, CSS, and JavaScript

## Objective

The objective of this project is to create a simple interactive web page using HTML, CSS, and JavaScript. The web page contains a greeting message, four color boxes, an input field to enter the user's name, and a button to greet the user. The project demonstrates the use of DOM manipulation and event handling in JavaScript.

## Project Description

This application displays a greeting message and allows the user to interact with different elements on the page.

The page contains:

* A heading that initially displays "Hello".
* Four boxes named Red, Blue, Green, and Yellow.
* An input field where the user can enter their name.
* A button to update the greeting message.

When the user enters a name and clicks the button, the greeting message changes to "Hello, [Name]". When any color box is clicked, its background color changes to the corresponding color.

## Features

### Greeting Functionality

* Displays "Hello" initially.
* Updates the heading to include the user's name.

### Color Box Functionality

* Red Box changes to red color.
* Blue Box changes to blue color.
* Green Box changes to green color.
* Yellow Box changes to yellow color.

### User-Friendly Interface

* Simple and clean layout.
* Easy interaction using buttons and clickable boxes.

## How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Open the `index.html` file in any web browser.
4. Enter your name in the input field.
5. Click the **Greet Me** button.
6. Click any color box to see the color change.

## Working of the Program

### Step 1

The webpage loads and displays:
Hello

### Step 2

The user enters a name in the input field.

Example:
Abhay

### Step 3

The user clicks the **Greet Me** button.

Output:
Hello, Abhay

### Step 4

The user clicks any color box.

Example:

* Click Red → Box becomes Red
* Click Blue → Box becomes Blue
* Click Green → Box becomes Green
* Click Yellow → Box becomes Yellow
